# Коды ошибок

Unne использует стандартизированные коды ошибок с префиксом `STATUS_UNNE_`.

## Справочник кодов ошибок

| Код | Описание |
|-----|----------|
| `STATUS_UNNE_AUTH_FAILED` | Недействительный или просроченный токен аутентификации |
| `STATUS_UNNE_SUBDOMAIN_TAKEN` | Запрашиваемый поддомен уже используется |
| `STATUS_UNNE_PROTOCOL_ERROR` | Ошибка протокола рукопожатия (некорректный запрос) |
| `STATUS_UNNE_TUNNEL_LIMIT` | Достигнут максимальный лимит туннелей для вашей учетной записи |
| `STATUS_UNNE_PROTOCOL_DENIED` | Вашей учетной записи не разрешено использовать этот протокол |
| `STATUS_UNNE_TRAFFIC_EXCEEDED` | Превышен лимит трафика за текущий период |
| `STATUS_UNNE_DEVICE_LIMIT` | Достигнут максимальный лимит устройств для вашей учетной записи |
| `STATUS_UNNE_USER_DISABLED` | Ваша учетная запись отключена администратором |
| `STATUS_UNNE_TOKEN_DISABLED` | Этот токен был отозван или отключен |
| `STATUS_UNNE_PORT_UNAVAILABLE` | Запрашиваемый TCP-порт недоступен |
| `STATUS_UNNE_SERVER_ERROR` | Внутренняя ошибка сервера |

## Страницы HTTP-ошибок

При доступе к туннелю через браузер сервер показывает стилизованные страницы ошибок:

| Страница | HTTP-статус | Когда отображается |
|----------|-------------|---------------------|
| **Предупреждение** | 200 | Первое посещение из браузера (требуется подтверждение) |
| **Не найдено** | 404 | Туннель/поддомен не существует |
| **Не в сети** | 502 | Туннель существует, но upstream недоступен |
| **Ошибка** | Различный | Общая ошибка с кодом и описанием |

## Устаревшие коды ошибок

Для обратной совместимости со старыми серверами:

| Устаревший | Новый код |
|------------|-----------|
| `UNNE_ERR_403` | `STATUS_UNNE_AUTH_FAILED` |
| `UNNE_ERR_102` | `STATUS_UNNE_SUBDOMAIN_TAKEN` |
| `UNNE_ERR_101` | `STATUS_UNNE_PROTOCOL_ERROR` |
