# Веб-инспектор

Веб-инспектор предоставляет интерфейс в стиле Network Console из Chrome DevTools для мониторинга трафика туннелей в вашем браузере.

## Включение

```bash
# С быстрым туннелем
unne http 3000 --webui

# С пользовательским портом
unne http 3000 --webui --webui-port 9090

# В конфигурации
gui:
  webui: true
  webui_port: 4040
```

Откройте `http://localhost:4040` в вашем браузере.

## Функции

- **Таблица запросов** -- Метод, статус, хост, путь, IP, длительность, имя туннеля
- **Панель деталей запроса** -- Изменяемая разделенная панель с заголовками и телом
- **Обновления в реальном времени** -- В реальном времени через WebSocket
- **Фильтр** -- Поиск по методу, пути, статусу, IP
- **Фильтр по туннелю** -- Показать запросы для конкретного туннеля
- **Группировка по IP** -- Переключение для группировки запросов по IP-адресу клиента
- **Форматирование JSON** -- Тело автоматически форматируется, если это валидный JSON
- **Очистка** -- Сброс списка запросов

## API-эндпоинты

Веб-инспектор предоставляет REST API:

| Метод | Путь | Описание |
|-------|------|----------|
| `GET` | `/api/requests` | Список перехваченных запросов |
| `GET` | `/api/requests?tunnel=name` | Фильтрация по туннелю |
| `GET` | `/api/requests/:id` | Получить детали запроса |
| `WS` | `/api/ws` | WebSocket для обновлений в реальном времени |
