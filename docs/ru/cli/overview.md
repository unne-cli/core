# Обзор Unne CLI

Unne CLI -- это клиентское приложение, которое создает туннели от вашей локальной машины к серверу Unne.

## Команды

| Команда | Описание |
|---------|----------|
| `unne setup` | Интерактивная первоначальная настройка |
| `unne http <port>` | Быстрый HTTP-туннель на локальном порту |
| `unne tcp <port>` | Быстрый TCP-туннель на локальном порту |
| `unne start` | Запуск туннелей, определенных в `unne.yml` |
| `unne check` | Проверка подключения к серверу |
| `unne version` | Показать информацию о версии |
| `unne help` | Показать справку |

## Глобальные флаги

| Флаг | Описание |
|------|----------|
| `--config <path>` | Путь к файлу конфигурации (по умолчанию: `unne.yml`) |
| `--subdomain <name>` | Желаемый поддомен (только для HTTP) |
| `--remote-port <port>` | Удаленный порт (только для TCP) |
| `--proxy <url>` | URL прокси (`socks5://` или `http://`) |
| `--skip-warning`, `--sw` | Пропустить страницу предупреждения в браузере |
| `--no-tui` | Отключить TUI, использовать только логи |
| `--webui` | Включить веб-инспектор |
| `--webui-port <port>` | Порт веб-инспектора (по умолчанию: `4040`) |

## Быстрые примеры

```bash
# Открыть доступ к локальному веб-приложению
unne http 3000

# Открыть доступ с пользовательским поддоменом
unne http 3000 --subdomain myapp

# TCP-туннель для PostgreSQL
unne tcp 5432 --remote-port 15432

# Запустить все туннели из конфигурации с веб-инспектором
unne start --webui

# Режим без интерфейса через прокси
unne http 8080 --no-tui --proxy socks5://127.0.0.1:1080
```
