# Что такое Unne?

Unne — это self-hosted туннельное решение, позволяющее открывать локальные сервисы в интернет через собственный сервер. Аналог ngrok, Cloudflare Tunnel или localtunnel, но на вашем сервере.

## Компоненты

| Компонент | Описание |
|-----------|----------|
| **Unne CLI** (`unne`) | Клиентское приложение для создания туннелей |
| **Unne Server** (`unns`) | Серверное приложение для приёма подключений и маршрутизации трафика |

## Как это работает

```
Интернет → Unne Server (ваш VPS) → Yamux → Unne CLI → Локальный сервис
```

1. Запускаете `unns` на VPS с публичным IP
2. DNS `*.yourdomain.com` указывает на сервер
3. Запускаете `unne http 3000` на локальной машине
4. Локальный сервис доступен по `https://random.yourdomain.com`

## Ключевые возможности

- **HTTP и TCP туннели**
- **TUI дашборд** с инспектором запросов
- **Веб инспектор** в стиле Chrome DevTools
- **Управление пользователями** с лимитами
- **Админ панель**
- **Поддержка прокси** (SOCKS5, HTTP)
- **Страницы предупреждений** для посетителей
