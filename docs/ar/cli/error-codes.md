# رموز الخطأ

يستخدم Unne رموز خطأ موحدة مسبوقة بـ `STATUS_UNNE_`.

## مرجع رموز الخطأ

| الرمز | الوصف |
|-------|-------|
| `STATUS_UNNE_AUTH_FAILED` | رمز مصادقة غير صالح أو منتهي الصلاحية |
| `STATUS_UNNE_SUBDOMAIN_TAKEN` | النطاق الفرعي المطلوب قيد الاستخدام بالفعل |
| `STATUS_UNNE_PROTOCOL_ERROR` | خطأ في بروتوكول المصافحة (طلب غير صحيح) |
| `STATUS_UNNE_TUNNEL_LIMIT` | تم الوصول إلى الحد الأقصى للأنفاق لحسابك |
| `STATUS_UNNE_PROTOCOL_DENIED` | لا يُسمح لحسابك باستخدام هذا البروتوكول |
| `STATUS_UNNE_TRAFFIC_EXCEEDED` | تم تجاوز حد حركة المرور للفترة الحالية |
| `STATUS_UNNE_DEVICE_LIMIT` | تم الوصول إلى الحد الأقصى للأجهزة لحسابك |
| `STATUS_UNNE_USER_DISABLED` | تم تعطيل حساب المستخدم الخاص بك من قبل المدير |
| `STATUS_UNNE_TOKEN_DISABLED` | تم إلغاء أو تعطيل هذا الرمز |
| `STATUS_UNNE_PORT_UNAVAILABLE` | منفذ TCP المطلوب غير متاح |
| `STATUS_UNNE_SERVER_ERROR` | خطأ داخلي في الخادم |

## صفحات خطأ HTTP

عند الوصول إلى نفق عبر المتصفح، يعرض الخادم صفحات خطأ منسّقة:

| الصفحة | حالة HTTP | متى |
|--------|-----------|-----|
| **تحذير** | 200 | الزيارة الأولى من المتصفح (يتطلب تأكيداً) |
| **غير موجود** | 404 | النفق/النطاق الفرعي غير موجود |
| **غير متصل** | 502 | النفق موجود لكن الخدمة المحلية غير قابلة للوصول |
| **خطأ** | متغير | خطأ عام مع رمز ووصف |

## رموز الخطأ القديمة

للتوافق مع الإصدارات القديمة من الخوادم:

| القديم | الرمز الجديد |
|--------|-------------|
| `UNNE_ERR_403` | `STATUS_UNNE_AUTH_FAILED` |
| `UNNE_ERR_102` | `STATUS_UNNE_SUBDOMAIN_TAKEN` |
| `UNNE_ERR_101` | `STATUS_UNNE_PROTOCOL_ERROR` |
