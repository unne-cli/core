# Unne CLI шолу

Unne CLI -- жергілікті компьютерден Unne серверіне туннельдер жасайтын клиенттік қосымша.

## Командалар

| Команда | Сипаттама |
|---------|-----------|
| `unne setup` | Бірінші рет интерактивті баптау |
| `unne http <port>` | Жергілікті порттағы жылдам HTTP туннель |
| `unne tcp <port>` | Жергілікті порттағы жылдам TCP туннель |
| `unne start` | `unne.yml` файлында анықталған туннельдерді іске қосу |
| `unne check` | Сервермен байланысты тексеру |
| `unne version` | Нұсқа туралы ақпарат |
| `unne help` | Анықтама көрсету |

## Жалпы жалаушалар

| Жалауша | Сипаттама |
|---------|-----------|
| `--config <path>` | Конфигурация файлының жолы (әдепкі: `unne.yml`) |
| `--subdomain <name>` | Қажетті субдомен (тек HTTP) |
| `--remote-port <port>` | Қашықтағы порт (тек TCP) |
| `--proxy <url>` | Прокси URL (`socks5://` немесе `http://`) |
| `--skip-warning`, `--sw` | Браузер ескерту бетін өткізіп жіберу |
| `--no-tui` | TUI-ді өшіру, тек журнал шығысы |
| `--webui` | Web инспекторды қосу |
| `--webui-port <port>` | Web инспектор порты (әдепкі: `4040`) |

## Жылдам мысалдар

```bash
# Жергілікті веб-қосымшаны ашу
unne http 3000

# Арнайы субдоменмен ашу
unne http 3000 --subdomain myapp

# PostgreSQL үшін TCP туннель
unne tcp 5432 --remote-port 15432

# Конфигурациядан барлық туннельдерді web инспектормен іске қосу
unne start --webui

# Прокси арқылы headless режим
unne http 8080 --no-tui --proxy socks5://127.0.0.1:1080
```
