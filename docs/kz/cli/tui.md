# TUI бақылау тақтасы

Unne CLI-де [BubbleTea](https://github.com/charmbracelet/bubbletea) негізіндегі кіріктірілген терминал UI бар.

## Орналасу

```
┌─────────────────────────────────────────────────────┐
│ UNNE TUNNEL  v2.0.0              ● ONLINE           │
│━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━│
│ ● web-app (12)  │  ○ api (3)  │  ◌ database        │
│─────────────────────────────────────────────────────│
│ #     Method  Path              Status    Time      │
│ ─────────────────────────────────────────────────── │
│ 1     GET     /api/users        200       12ms      │
│ 2     POST    /api/login        201       45ms      │
│ 3     GET     /static/app.js    304       2ms       │
│►4     DELETE  /api/users/5      500       120ms     │
│                                                     │
│ ↑↓ navigate  enter details  esc back  v split  q quit│
└─────────────────────────────────────────────────────┘
```

## Перне тіркесімдері

| Перне | Әрекет |
|-------|--------|
| `↑` / `k` | Сұраныстар тізімінде жоғары жылжу |
| `↓` / `j` | Сұраныстар тізімінде төмен жылжу |
| `Tab` | Келесі туннель қойындысы |
| `Shift+Tab` | Алдыңғы туннель қойындысы |
| `Enter` | Сұраныс мәліметтерін көру / Сұраныс↔Жауап ауысу |
| `Esc` | Тізімге қайту / мәліметтер панелін жабу |
| `v` | Тік/көлденең бөлуді ауыстыру |
| `q` / `Ctrl+C` | Шығу |

## Көп туннельді қойындылар

Бірнеше туннель іске қосылғанда (`unne start` арқылы), әрбір туннельдің өз қойындысы болады:
- Қосылыс күйінің индикаторы (`●` қосылған, `◌` қосылуда, `○` офлайн)
- Туннель атауы мен сұраныстар саны
- Тәуелсіз сұраныстар тізімі

## Сұраныс мәліметтерін көру

Көру үшін сұраныста `Enter` басыңыз:
- **Сұраныс қойындысы**: Әдіс, жол, хост, тақырыптар, дене
- **Жауап қойындысы**: Күй коды, ұзақтық, тақырыптар, дене

Сұраныс пен Жауап арасында ауысу үшін `Enter` басыңыз.

## Бөлінген көрініс

Ауысу үшін `v` басыңыз:
- **Тік бөлу**: Сұраныстар тізімі сол жақта, мәліметтер оң жақта
- **Көлденең бөлу**: Сұраныстар тізімі жоғарыда, мәліметтер төменде

## TUI-ді өшіру

Headless/CI орталары үшін:

```bash
unne start --no-tui
```

Журналдар stdout-қа қарапайым мәтін форматында шығарылады.
