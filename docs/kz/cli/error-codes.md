# Қате кодтары

Unne `STATUS_UNNE_` префиксімен стандартталған қате кодтарын қолданады.

## Қате кодтарының анықтамалығы

| Код | Сипаттама |
|-----|-----------|
| `STATUS_UNNE_AUTH_FAILED` | Жарамсыз немесе мерзімі өткен аутентификация токені |
| `STATUS_UNNE_SUBDOMAIN_TAKEN` | Сұралған субдомен бос емес |
| `STATUS_UNNE_PROTOCOL_ERROR` | Қол алысу протоколының қатесі (дұрыс емес сұраныс) |
| `STATUS_UNNE_TUNNEL_LIMIT` | Тіркелгіңіздің туннель лимитіне жеттіңіз |
| `STATUS_UNNE_PROTOCOL_DENIED` | Тіркелгіңізге бұл протоколды қолдануға рұқсат жоқ |
| `STATUS_UNNE_TRAFFIC_EXCEEDED` | Ағымдағы кезеңнің трафик лимиті асырылды |
| `STATUS_UNNE_DEVICE_LIMIT` | Тіркелгіңіздің құрылғы лимитіне жеттіңіз |
| `STATUS_UNNE_USER_DISABLED` | Пайдаланушы тіркелгіңіз әкімші тарапынан өшірілген |
| `STATUS_UNNE_TOKEN_DISABLED` | Бұл токен қайтарып алынған немесе өшірілген |
| `STATUS_UNNE_PORT_UNAVAILABLE` | Сұралған TCP порт қолжетімсіз |
| `STATUS_UNNE_SERVER_ERROR` | Ішкі сервер қатесі |

## HTTP қате беттері

Браузер арқылы туннельге кіргенде, сервер стильденген қате беттерін көрсетеді:

| Бет | HTTP күйі | Қашан |
|-----|-----------|-------|
| **Ескерту** | 200 | Браузерден бірінші кіру (растау қажет) |
| **Табылмады** | 404 | Туннель/субдомен жоқ |
| **Офлайн** | 502 | Туннель бар, бірақ upstream қолжетімсіз |
| **Қате** | Әр түрлі | Кодпен және сипаттамамен жалпы қате |

## Ескі қате кодтары

Ескі серверлермен үйлесімділік үшін:

| Ескі | Жаңа код |
|------|----------|
| `UNNE_ERR_403` | `STATUS_UNNE_AUTH_FAILED` |
| `UNNE_ERR_102` | `STATUS_UNNE_SUBDOMAIN_TAKEN` |
| `UNNE_ERR_101` | `STATUS_UNNE_PROTOCOL_ERROR` |
